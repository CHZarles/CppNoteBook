📋 项目简介

这是一个使用 C++ 实现的 CGI (Common Gateway Interface) 程序演示项目。项目展示了如何：

  • 使用 C++ 编写 CGI 程序
  • 配置 Nginx 支持 CGI
  • 处理 HTTP GET 和 POST 请求
  • 解析表单数据和查询参数
  • 生成动态 HTML 内容

🎯 Hello CGI

基础的 CGI 程序，显示服务器信息和环境变量

📝 表单处理

演示如何处理 HTML 表单提交的数据

🔧 自动构建

使用 Makefile 管理项目编译和部署

🎉 Hello CGI 📝 测试表单

🛠️ 技术栈

C++ CGI Nginx HTML5 Linux Makefile

🚀 快速开始

  1. 编译项目：make
  2. 安装部署：make install
  3. 启动 Nginx：sudo systemctl start nginx
  4. 访问本页面测试功能
